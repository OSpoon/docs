---
title: "Introduction"
description: "LangChain 大模型应用开发"
---

1. LangChain 是一个基于大型语言模型的应用程序开发框架 (LLMs).
2. LangChain 简化了 LLM 应用程序生命周期中的每一个阶段：
   1. 开发：使用 LangChain 的开源组件和第三方集成来构建您的应用程序。使用 LangGraph 构建具有一流流式传输和人工介入支持的状态化代理。
   2. 生产：使用 LangSmith 检查、监控和评估您的应用程序，以便您可以持续优化并有信心部署。
   3. 部署：使用 LangGraph 平台将您的 LangGraph 应用程序转换为生产就绪的 API 和助手。
3. LangChain 实现了一个标准接口，用于大型语言模型及相关技术，如嵌入模型和向量存储，并与数百个提供商集成。
